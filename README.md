## Purpose
The Blood Donation Application aims to create a user-friendly platform that facilitates blood donation activities. It connects donors with those in need of blood, promoting a seamless and efficient donation process.

## Key Features
- User registration and authentication (Donor, Admin, Volunteer roles)
- Blood donation request management
- Donor management
- Content management
- Role-based access control
- User profiles with update functionality
- Dashboard for each user role (Donor, Admin, Volunteer)
- Public pages for Home, Login, Registration, Search, and Donation Requests
- Stripe payment integration for funding
- JWT for API protection
- Responsive UI design

## Technologies Used
- **Frontend:** React
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** Firebase
- **Payment Gateway:** Stripe
- **Other:** ImageBB for avatar uploads, environment variables for sensitive data.
